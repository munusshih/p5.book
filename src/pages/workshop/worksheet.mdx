# make a book with code

You already know how to draw with p5.js. Now let's use that to make a real, printable book — every page drawn by code, exported as a PDF.

p5.book adds one idea on top of p5.js: **`draw()` runs once per page.** By the time it finishes, you have a complete PDF.

---

## 0 — add p5.book

Open your sketch in the **p5.js editor**. Click the **▼** arrow next to your sketch name, then click **index.html**.

Find the `</body>` tag near the bottom. Add these two lines right _before_ it:

```html
<script src="https://unpkg.com/jspdf@latest/dist/jspdf.umd.min.js"></script>
<script src="https://cdn.jsdelivr.net/gh/munusshih/p5.book@main/p5.book.js"></script>
```

Click back on **sketch.js** to return to your code.

<div class="note">

jsPDF generates the PDF file. p5.book connects your p5 sketch to it.

</div>

## 1 — your first book

Replace your sketch code with this:

```js
let book;

function setup() {
  book = createBook(4, 6, 8); // 4 in wide, 6 in tall, 8 pages
  // canvas is created automatically at the right aspect ratio
  // book.setDPI(300); ← uncomment for print-quality export
}

function draw() {
  background(255);
  circle(width / 2, height / 2, 100);

  book.addPage(); // saves this frame as a page — always call this last
}
```

<div class="step">

Press **▶ Play**. The sketch runs 8 times, once per page. After the last page a viewer opens — try the view dropdown at the top left (flipbook / grid / 3D view), and the download button.

</div>

<div class="note">

`book.addPage()` at the end of `draw()` says: "save everything on the canvas right now as a page, then move on to the next."

</div>

<div class="try">

Change the page count: try 4, 12, or 20. The viewer should show however many pages you set.

What happens if you uncomment `book.setDPI(300)` in `setup()`? (The canvas stays the same on screen but the PDF exports at higher resolution.)

</div>

## 2 — which page am I on?

p5.book gives you three numbers to know where you are:

| | |
|---|---|
| `book.pageNumber` | current page — starts at **1** |
| `book.page` | same, starts at **0** — useful as an array index |
| `book.totalPages` | total pages passed to `createBook()` |

<div class="step">

Add a page counter to your sketch:

```js
function draw() {
  background(255);

  fill(0);
  textSize(20);
  text(book.pageNumber + " / " + book.totalPages, 20, 40);

  book.addPage();
}
```

</div>

You can use `book.pageNumber` to make things _change_ per page:

```js
function draw() {
  background(255);

  // circle moves right each page: 40, 80, 120…
  let x = book.pageNumber * 40;
  circle(x, height / 2, 60);

  book.addPage();
}
```

<div class="try">

What else could you change using `book.pageNumber`? Try the circle's size, the background brightness, or the y position.

</div>

## 3 — gradual change

`book.progress` is the easiest way to make things evolve smoothly across the whole book.

It's **0** on the first page and **1** on the last — multiply it or pass it to `lerp()`.

```js
// background fades from black to white
let brightness = book.progress * 255;
background(brightness);
```

<div class="step">

```js
function draw() {
  let t = book.progress; // 0 on page 1 → 1 on last page

  background(255 - t * 255); // fades white → black

  let sz = lerp(10, 400, t);
  fill(255);
  circle(width / 2, height / 2, sz); // starts tiny, ends huge

  book.addPage();
}
```

Press **▶ Play** and watch the progression across pages.

</div>

<div class="try">

Try:
- Fade between two colours: `background(lerp(220, 40, t), lerp(80, 180, t), lerp(200, 80, t))`
- Drift a shape across: `circle(lerp(40, width - 40, t), height / 2, 60)`

</div>

## 4 — cover and back cover

Use `isFirstPage()` and `isLastPage()` to give special treatment to the cover and back cover.

<div class="step">

```js
function draw() {
  if (book.isFirstPage()) {
    background(0); // black cover
    fill(255);
    textSize(32);
    textAlign(CENTER, CENTER);
    text("my book", width / 2, height / 2);
  } else if (book.isLastPage()) {
    background(0);
    fill(255);
    textSize(16);
    textAlign(CENTER, CENTER);
    text("the end", width / 2, height / 2);
  } else {
    background(255);
    fill(0);
    textSize(20);
    text(book.pageNumber, 20, 40);
  }

  book.addPage();
}
```

</div>

<div class="try">

Design your own cover — try a different colour, add a shape, put your name on it.

</div>

## 5 — bleed

**Bleed** is extra image that extends past the cut line. If your background colour should go right to the edge of the printed page — not stop before the trim — you need bleed.

<div class="step">

Add bleed in `setup()` and draw backgrounds on `book.bleed`:

```js
function setup() {
  book = createBook(4, 6, 8);
  book.setBleed(0.125); // 1/8 inch — US standard (use 3, "mm" for Europe)
}

function draw() {
  // book.bleed covers the full bleed area — draw backgrounds here
  book.bleed.background(220, 60, 80);

  // the main canvas sits on top, inside the trim marks
  clear(); // transparent so bleed shows through

  fill(255);
  textSize(20);
  textAlign(LEFT, TOP);
  text(book.pageNumber, 20, 24);

  book.addPage();
}
```

Press **▶ Play** and notice in the viewer that the pink extends to the trim marks.

</div>

<div class="note">

`book.bleed` is a separate drawing layer sized to the full bleed area. Use `clear()` instead of `background()` on the main canvas so the bleed layer shows through behind your content.

</div>

<div class="try">

Fade the bleed background across pages:

```js
let c = lerpColor(color(220, 60, 80), color(40, 80, 200), book.progress);
book.bleed.background(c);
```

</div>

## 6 — make your book

Put it all together and make something you'd actually want to print.

**Ideas:**
- **Colour book** — each page a different random hue in HSB mode
- **Word book** — an array of words, one word fills each page in big type
- **Shape book** — a circle slowly turns into a rectangle over 10 pages
- **Counting book** — the page number drawn huge and centred, one colour per page
- **Name book** — each letter of your name on its own page, alternating black / white
- **Mood book** — each page a different texture, palette, or abstract composition

<div class="try">

Build your book. Click **download** in the viewer when you're done.

Try the **3D view** dropdown to see your book spinning as a physical object. Download → **Saddle Stitch**, print double-sided, fold in half — you have a real zine.

</div>

## cheat sheet

| | |
|---|---|
| **in setup() — run once** | |
| `createBook(4, 6, 8)` | 4 in wide, 6 in tall, 8 pages — canvas auto-created |
| `createBook("A5", 8)` | standard paper size (A3 A4 A5 A6 letter legal tabloid) |
| `book.setDPI(300)` | high-resolution export — for printing |
| `book.setBleed(0.125)` | 1/8 inch bleed — or `(3, "mm")` for European standard |
| **in draw() — run once per page** | |
| `book.pageNumber` | current page — starts at 1 |
| `book.page` | same, starts at 0 — use as an array index |
| `book.totalPages` | total pages |
| `book.progress` | 0 on page 1 → 1 on last page |
| `book.isFirstPage()` | true only on the first page |
| `book.isLastPage()` | true only on the last page |
| `book.bleed.background(c)` | full-bleed background — extends past trim |
| `book.addPage()` | save this page — always put this **last** |
| **in the viewer** | |
| `← / →` | previous / next page |
| `[ / ]` | jump to first / last page |
| `?` | keyboard shortcuts |
| view dropdown | flipbook · grid · 3D view |
| download dropdown | PDF · Saddle Stitch · Cover · Frames |
